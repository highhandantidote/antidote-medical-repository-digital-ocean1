Direct File Update Method

If you prefer to directly replace the logo file:

  1. Create an SVG file named antidote-logo.svg
  2. Upload it to /static/images/ directory
  3. Update the reference in templates/base.html to point to your SVG file

Change this line in base.html:

<img src="{{ '{{' }} url_for('static', filename='images/antidote-logo-new.png') {{ '}}' }}" alt="Antidote" height="60" class="me-2">

To this:

<img src="{{ '{{' }} url_for('static', filename='images/antidote-logo.svg') {{ '}}' }}" alt="Antidote" height="60" class="me-2">
